# MapToPoster Studio

Local webapp for generating posters with `create_map_poster.py`.

From the repository root, start it with:

```bash
./start.sh
```

Or run the server directly:

```bash
.venv/bin/python webapp/app.py 127.0.0.1 8080
```

Open `http://127.0.0.1:8080`, choose city, theme, size, format, and download
the generated poster. Files are saved in `posters/`.
