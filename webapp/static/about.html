Local graphical interface

Urban maps turned into posters, without using the command line.

This webapp lets you choose city, theme, paper size, output format, and fonts from a visual interface. It generates previews, saves PDF, PNG, or SVG files, and can create one ZIP with every theme.

Start creating See examples

Where it comes from

This project is simply a graphical interface built on top of the original originalankur/maptoposter. The goal is to make poster generation more immediate while keeping the core idea intact: turning geographic city data into a printable composition.

What you can do

You can change city, distance, proportions, A0/A1/A3/A4 presets, visual theme, and final format. The preview regenerates while you edit settings, so you immediately understand what kind of poster you are preparing.

How it works

The webapp uses local caches for cities and previews whenever possible, making repeated edits faster. When a city cannot be found, you can enter latitude and longitude manually.
